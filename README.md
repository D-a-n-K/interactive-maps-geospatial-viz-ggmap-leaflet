# lab-08-starter
This is the starter repository for Lab 8: Data Visualization and Collaborative Coding in R and Python in Professor Cogburn's course ITEC/SIS 724 Advanced Text Analytis with AI and ML. This lab is a little different from previous labs in that it focuses on data visualization and collaborative coding using R and Python. The collaborative coding aspect will involve using GitHub for more than just to manage and share code, but to teach you how to work collaboratively on coding projects. These skills include learning how to create and manage branches, resolve merge conflicts, and conduct code reviews.
